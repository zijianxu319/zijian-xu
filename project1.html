Project 1: Restaurant Website Design

Project Background

This project aimed to create a restaurant menu and an online ordering page. The design focused on creating an intuitive and visually appealing website. By using HTML, CSS, and JavaScript, I built a responsive design that ensures the website is functional on both desktop and mobile devices.

Technical Details

  • Built with HTML, CSS, and JavaScript for a responsive and modern design.
  • Used media queries to ensure the website works well on a variety of devices.
